## 설문 조사 후보

### 항목

- 사이트에서 가장 많이 보는 컨텐츠는 무엇인가?
- 더 추가되었으면 하는 컨텐츠는 무엇인가?
- 원하는 자료를 사이트에서 찾기 수월한가?
- 어떻게 우리 사이트를 알고 방문하게 되었나?
- OSE의 사용 목적은?
- Contribution하는데 어려움이 있는가?
- 현재 OSE를 사용하는 곳이 있는지? 없다면 향후 적용할 예정인지?
- 현재 OSE에서 제공하는 기능 (API) 이외에 추가되었으면 기능이 있는지?
